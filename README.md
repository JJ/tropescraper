# tropescraper

A scraper for the website TV tropes.

## Install

Install all dependencies with:

    pip install -e .
    
(pip should be installed and available).

## Run

Run it with

    bin/scrape-tvtropes
    
It will take a good while while it scrapes ~12k films.
